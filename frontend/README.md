# ☕ Coffee Cafe Website

A modern and fully responsive Coffee Cafe website built using React.js, Tailwind CSS, and Vite. This project focuses on delivering a clean UI, smooth user experience, and fast performance.

---

## 🚀 Live Demo

🔗 https://www.the-coffeecafe.netlify.app/

---

## 📁 Repository Structure

Coffee_Cafe/
│── frontend/   → React application
│── README.md

---

## 🛠️ Tech Stack

* React.js
* Tailwind CSS
* Vite
* AOS (Animate On Scroll)

---

## ✨ Features

* Fully responsive design (Mobile, Tablet, Desktop)
* Clean and modern UI
* Smooth navigation and interactive UI elements
* Scroll animations using AOS
* Optimized performance with Vite
* Reusable and modular components

## ⚡ Installation & Setup

### 1. Clone the repository

```bash
git clone https://github.com/Kavita9-git/Coffee_Cafe.git
```

### 2. Navigate to project folder

```bash
cd Coffee_Cafe/frontend
```

### 3. Install dependencies

```bash
npm install
```

### 4. Run the project

```bash
npm run dev
```
